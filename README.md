# Dotfiles

nyogjtrc's dotfiles. managed by Homemaker.

---

## Install dotfiles

### Get dotfiles repo

```
$ git clone git@github.com:nyogjtrc/dotfiles.git .dotfiles
```

### Install homemaker

```
$ ./installation/homemaker.sh
```

### Install dotfiles via Homemaker

```
$ ./dotfiles.sh
```
